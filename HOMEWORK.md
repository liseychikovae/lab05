# Домашнее задание lab05

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы V: изучение фреймворков для тестирования на примере gtest.

## Выполненные пункты

- тест `FormatterEx.WrapsText` проверяет, что расширенный formatter сохраняет исходный текст;
- тест `Solver.FindsTwoRoots` проверяет квадратное уравнение с двумя корнями;
- тест `Solver.RejectsLinearEquation` проверяет обработку неверного коэффициента `a = 0`; 
- тесты встроены в CMake и могут запускаться локально или в CI.

## Как проверить домашнее задание

```bash
cmake -S . -B _build -DCMAKE_BUILD_TYPE=Release -DBUILD_TESTS=ON
cmake --build _build
ctest --test-dir _build --output-on-failure
./_build/tests/check
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
